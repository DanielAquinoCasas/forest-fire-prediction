# 🔥 Forest Fire Prediction Using Weather Conditions

## 📖 Project Overview

This project investigates whether weather conditions can help explain the size of forest fires.

Using Python, exploratory data analysis (EDA), and multiple linear regression, the project analyzes how temperature, humidity, wind speed, and rainfall influence the burned area of forest fires.

The objective is to identify meaningful relationships between environmental variables and wildfire size while demonstrating a complete data analytics workflow.

---

## 🎯 Business Problem

Wildfires have significant environmental and economic impacts.

Understanding how weather conditions influence wildfire size can help support planning, prevention, and resource allocation.

This project explores whether historical weather measurements can be used to explain the variation in burned area.
